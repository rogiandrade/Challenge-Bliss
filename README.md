The technologies used were:
- React.JS
- TypeScript
- Vite.JS
- VSCode
- Axios

<h1> ℹ️ Getting Started </h1>

First, run for install dependencies:

```bash
yarn
# or
npm install
```

Second, run for start server:

```bash
npm run dev
```

Open http://127.0.0.1:5173/ with your browser to see the result.
